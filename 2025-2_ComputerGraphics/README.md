# OpenGL Practice
In this course, we learned the basic concepts of **Computer Graphics**.
This project contains my practice work using `OpenGL` with `GLEW`+`GLFW`.

There are a total of 6 assignments.
Please check `./Assignment_Results` directory for the results.

## Development Environment
- Windows
- Visual Studio 2022

## Dependencies
- [GLEW](https://glew.sourceforge.net/)
- [GLFW](https://github.com/glfw/glfw)
- [GLM](https://github.com/g-truc/glm)
- [stb_image](https://github.com/nothings/stb/blob/master/stb_image.h)
